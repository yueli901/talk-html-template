What this template gives you

  • Pure static — hosts free on GitHub Pages
  • Works offline — installable, no internet after first load
  • 1-D navigation — any arrow steps the talk
  • Slide numbers, no progress bar
  • Speaker view with notes & timer
  • Offline math via KaTeX
  • Draw on slides (chalkboard)
  • Menu / overview to jump anywhere
  • Floating PDF download + QR
  • Narration playback from object storage

Everything below is a live demo of one feature. Navigate with the arrows.

Navigating (1-D / linear)

Every direction moves one step:

  • → ↓ Space — next
  • ← ↑ — previous
  • Esc — overview grid
  • M or the ☰ button — menu / table of contents

For presenting:

  • S — speaker view (notes + next + timer)
  • F — fullscreen · B/. — black
  • Alt+click — zoom into a figure
  • Swipe on tablets · deep-link every slide via the URL #/n

Step-by-step reveals (fragments)

Reveal points one at a time — good for building an argument:

  • First this idea appears…
  • then this one…
  • tip fragments can also highlight, grow, or fade.

Press the arrow again — the deck advances only when the fragments run out.

Use fragments sparingly; one idea per click.

Narration playback

Pre-record audio, host it on object storage, list it in a manifest — the page plays it and auto-advances.

  • the ▶ Play narrated button (bottom-left) appears only if audio exists
  • one clip per slide → re-record any slide independently
  • no recording in the browser; the page only loads and plays

This demo ships 3 sample clips — press ▶ Play narrated.


// narration/manifest.json
{
  "base": "https://cdn.example.com/talk/",
  "clips": { "0":"s00.m4a", "1":"s01.m4a", "2":"s02.m4a" }
}

Object-storage bucket must allow CORS from this origin.

Reuse this template

  1. Click Use this template on GitHub → new repo
  2. Edit the <section> slides in index.html
  3. Drop figures in assets/, set the QR to your URL
  4. Enable Settings → Pages → main / root
  5. (optional) add narration clips + narration/manifest.json
